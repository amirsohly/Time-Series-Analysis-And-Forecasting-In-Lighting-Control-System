Requirements :
(It is assumed that you have installed npm)

1) Django :

py -m pip install Django

2) Vue JS :

A) install vue.js :

    npm init vue@latest

checking update :

    npm update -g @vue/cli
         
OR
     
    yarn global upgrade --latest @vue/cli

B) install axios :

    npm install axios
         
OR
     
    yarn add axios


C) install chart.js :
     
    npm install chart.js
